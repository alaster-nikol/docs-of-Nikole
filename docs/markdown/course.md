# Как добавить свой курс на GetCourse

**Курс (тренинг) на GetCourse** — образовательный контент, который вы можете разместить в аккаунте своей онлайн-школы в любом формате: текстовый, видео, аудио или просто в виде файла.

Инструменты платформы позволяют создавать гибкую структуру тренинга с помощью уроков и подтренингов (аналог папки в папке), регулировать доступ к ним и создавать полноценный процесс обучения с помощью расписания уроков и заданий.

## Как создать тренинг в аккаунте

Создать тренинг могут:

* владелец аккаунта,
* [администратор](https://getcourse.ru/blog/1193620).

Для этого:

1. Перейдите в раздел «Обучение» — «Тренинги» и нажмите «Добавить тренинг».

    <img src="../images/480.png" class="border" alt="Как создать курс">

2. Введите название вашего курса.

3. Укажите описание (по желанию). Оно будет отображаться в списке тренингов.

4. Добавьте превью-картинку (по желанию). Она будет отображаться в списке тренингов.

5. Нажмите «Создать».

    <img src="../images/189.png" class="border" alt="Создание курса">

## Как настроить структуру курса и добавить контент

Чтобы организовать структуру вашего курса, добавляйте уроки и подтренинги в нужной последовательности.

<img src="../images/424.png" class="border" alt="Структура курса">

### Как добавить урок

**Уроки** — основные объекты тренинга, в которых будет находиться ваш обучающий материал (тексты, видео, файлы, задания и т. д.).

Добавить урок в тренинг могут:

* владелец аккаунта,
* [администратор](https://getcourse.ru/blog/1193620),
* [сотрудник-преподаватель](https://getcourse.ru/blog/451908).

Для этого:

1. Откройте тренинг, нажмите «Добавить» и выберите «Урок».

    <img src="../images/240.png" class="border" alt="Как создать урок">

2. Введите название урока.

3. Заполните описание (по желанию).

4. Выберите тип урока:
    * **Пустой шаблон** — урок с новым удобным конструктором. Выбирайте шаблонные блоки для добавления любого материала и редактируйте по своему усмотрению.
    * **Вебинар** — урок с возможностью запустить прямую трансляцию для учеников тренинга.

5. Нажмите «Создать».

    <img src="../images/212.png" class="border" alt="Создание урока">

После создания откроется редактор урока, в котором вы можете разместить любые обучающие материалы (тексты, изображения, файлы, видео и т.д.).

<img src="../images/290.png" class="border" alt="Редактор урока">

### Как добавить подтренинг (модуль)

**Подтренинг** — отдельная папка или модуль с уроками внутри вашего тренинга.

Используйте подтренинги, когда хотите:

* визуально разделить основной тренинг на несколько частей по темам;
* выдавать доступ к части уроков только пользователям определенного тарифа;
* вынести в отдельную папку дополнительные/бонусные материалы;
* настроить последовательный доступ к группам уроков.

!!! warning "Важно"
    Доступ к каждому подтренингу нужно настраивать отдельно внутри самого подтренинга на вкладке «Доступ». Если настроить доступ только в основном тренинге, а в подтренинге оставить стандартный способ по покупке, то ученики не увидят такой подтренинг после покупки основного тренинга.

    Подробнее в инструкции [«Как настроить доступ к тренингу или подтренингу»](https://getcourse.ru/blog/276090).

<img src="../images/143.png" class="border" alt="Пример курса с подтренингами">

Добавить подтренинг могут:

* владелец аккаунта,
* [администратор](https://getcourse.ru/blog/1193620).

Для этого:

1. Откройте тренинг, нажмите «Добавить» и выберите «Подтренинг».

    <img src="../images/160.png" class="border" alt="Как создать подтренинг">

2. Введите название подтренинга.

3. Заполните описание (по желанию).

4. Выберите, куда будет добавлен подтренинг. По умолчанию выбран тренинг/подтренинг, из которого вы перешли к добавлению.

5. Добавьте превью-картинку, которая будет отображаться для подтренинга в общем списке уроков (по желанию).

6. Нажмите «Создать».

    <img src="../images/166.png" class="border" alt="Создание подтренинга">

После создания откроется подтренинг с аналогичными настройками, как и в тренинге, т. е. при необходимости вы можете настроить в нем отдельное расписание, доступ и т. д.

<img src="../images/181.png" class="border" alt="Тренинг и подтренинг в аккаунте">

!!! note "Обратите внимание"
    Максимальный уровень вложенности подтренингов — 10. Добавить/копировать/перенести подтренинг на 11 уровень нельзя.

<img src="../images/228.png" class="border" alt="Максимальная вложенность - 10">
